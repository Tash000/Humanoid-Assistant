# Humanoid Assistant - Expressive Head 
 
**Project Overview** 
This repository hosts the firmware and testing tools for a facially expressive humanoid robot head. 
 
**Current Achievements** 
* **Hardware Verification:** Implemented basic motor sweep tests to validate servo connections. 
* **Boundary Realization:** Developed a joystick-based jig system to establish safe movement boundaries. 
* **Expression Control:** Created a Serial Command Interface for specific facial expressions (jaw, eyes, brows, smile).
